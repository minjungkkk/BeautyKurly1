이벤트 롤링배너로 이동할 html 페이지 만들기


| 구분                        | 구성/설명                                    | 고정 px |
| ------------------------- | ---------------------------------------- | ----- |
| **Header**                | 상단 고정 영역 (메뉴, 로고, 아이콘)                   | 100   |
| **Content**               | Header 아래 ~ Footer Wrap 바로 위             | 1,200 |
| **Footer Wrap (텍스트 콘텐츠)** | 최상단 패딩 ~ 카피라이트까지                         | 136   |
| **Footer Safe Area**      | Footer Wrap 하단 여백 (Bottom Navigation 대응) | 80    |
| **Bottom Navigation**     | 하단 고정 네비게이션                              | 56    |
| **Footer Wrap 총 높이 기준**   | 텍스트 영역 + Safe Area                       | 216   |


┌───────────────────────┐
│ Header               │ 100px
├───────────────────────┤
│ Content              │ 1,200px
├───────────────────────┤
│ Footer Wrap Text      │ 136px
│ (메뉴, 정보, 카피라이트) │
├───────────────────────┤
│ Footer Safe Area      │ 80px
├───────────────────────┤
│ Bottom Navigation     │ 56px
└───────────────────────┘

Header는 100px로 고정되며, 페이지 상단에 항상 위치한다.

Content 영역은 1,200px이며, 페이지별 내용에 따라 스크롤 가능하다.

Footer Wrap 내부 텍스트 콘텐츠 영역은 136px로 구성되며, 하단 카피라이트까지 포함한다.

Footer Safe Area 80px을 확보하여, 하단 고정 네비게이션(56px)에 의해 콘텐츠가 가려지지 않도록 한다.

Footer Wrap 총 높이 기준은 216px이다.
